# PYNQ-Z2 FPGA Projects

FPGA projects implemented on the PYNQ-Z2 development board using Verilog HDL and Vivado.

## 🚀 Overview

This repository contains FPGA-based hardware projects focused on:
- OLED interfacing
- Display systems
- Embedded FPGA applications
- Real-time hardware control

## 🧰 Hardware Used
- PYNQ-Z2 FPGA Board
- SSD1331 OLED Display (PmodOLEDrgb)
- PmodKYPD Keypad

## 🛠️ Tools Used
- Verilog HDL
- Vivado
- FPGA Simulation & Verification

## 📂 Included Projects
- OLED Display Systems
- Image Rendering on OLED
- Traffic Light Simulation
- ALU Display Integration

## 📚 Learning Approach

Projects were developed through self-learning, experimentation, FPGA documentation, waveform debugging, and AI-assisted workflows.

- ChatGPT
- GitHub Copilot
