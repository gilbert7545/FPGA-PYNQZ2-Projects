# Image Inversion

Hardware-accelerated image inversion project implemented on the PYNQ-Z2 FPGA platform.

## Features
- Verilog-based image inversion
- AXI4-Lite IP integration
- Python-controlled hardware interaction
- FPGA image processing workflow

## Included
- Verilog image inverter module
- AXI4-Lite slave interface
- Python integration
- Hardware-software communication

## Tools Used
- Verilog HDL
- Python
- Vivado
- PYNQ Framework
